# Code Quality Rubric

Evaluate the generated code based on these criteria:

## Criteria

1. **Correctness** (40 points)
   - Code accomplishes the requested task
   - No syntax errors
   - Logic is sound

2. **Code Style** (30 points)
   - Follows language best practices
   - Consistent formatting
   - Meaningful variable/function names

3. **Completeness** (30 points)
   - All requirements addressed
   - Proper exports if needed
   - Type annotations where appropriate

## Scoring

- 90-100: Excellent - Production ready code
- 70-89: Good - Minor improvements needed
- 50-69: Acceptable - Functional but needs work
- Below 50: Needs Improvement - Does not meet requirements
